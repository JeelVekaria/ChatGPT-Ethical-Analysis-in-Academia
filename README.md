# ChatGPT-Ethical-Analysis-in-Academia 
Collaborative research project by myself and a team of four students, exploring the application of ChatGPT in academic environments, specifically within universities and colleges. Our data collection involved surveys, retrieving insights from 100 submissions. Used Python with Pandas, Numpy, and Matplotlib for data visualization, we processed and analyzed the data.
